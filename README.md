# Chaplman.nvim

Please read help for details.

"Chaplman" stands for "**Cha**tyusha **Pl**gin **Man**ager", and it is a plugin manager for vim/neovim.

## Requirements
  
  - Vim(version 8.0 or later) or Neovim
  - bash or zsh

## Features

  - You can write the configuration file with json.
